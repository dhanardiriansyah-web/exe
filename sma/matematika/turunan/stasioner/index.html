Bagian 1: Ringkasan Materi

1.1 Kemonotonan Fungsi (Fungsi Naik & Turun)

Kemonotonan sebuah fungsi menjelaskan di interval mana grafik fungsi tersebut bergerak naik atau turun jika dilihat dari kiri ke kanan. Analisis ini sangat bergantung pada turunan pertama fungsi, $f'(x)$.

  • Fungsi $f(x)$ dikatakan naik pada suatu interval jika untuk setiap $x$ di interval tersebut, $f'(x) > 0$.
  • Fungsi $f(x)$ dikatakan turun pada suatu interval jika untuk setiap $x$ di interval tersebut, $f'(x) < 0$.

1.2 Titik Stasioner dan Nilai Ekstrim

Titik stasioner adalah titik di mana gradien kurva sama dengan nol ($f'(c) = 0$). Titik ini merupakan kandidat untuk nilai maksimum atau minimum lokal.

Uji Turunan Pertama

  • Perubahan tanda $f'(x)$ dari (+) ke (-) di sekitar $c$ menandakan maksimum lokal.
  • Perubahan tanda $f'(x)$ dari (-) ke (+) di sekitar $c$ menandakan minimum lokal.
  • Jika tanda tidak berubah, maka $(c, f(c))$ adalah titik belok horizontal.

Uji Turunan Kedua

  • Jika $f''(c) < 0$, maka $f(c)$ adalah nilai balik maksimum lokal.
  • Jika $f''(c) > 0$, maka $f(c)$ adalah nilai balik minimum lokal.

Bagian 2: Contoh Soal & Pembahasan

1. (Pilihan Ganda Biasa)
Tentukan interval di mana fungsi $f(x) = x^3 - 3x^2 - 9x + 1$ turun.
Teori: Fungsi turun saat $f'(x) < 0$.
Langkah 1: Cari turunan pertama. $f'(x) = 3x^2 - 6x - 9$.
Langkah 2: Selesaikan pertidaksamaan $f'(x) < 0$.
$3x^2 - 6x - 9 < 0 \implies x^2 - 2x - 3 < 0 \implies (x-3)(x+1) < 0$.
Langkah 3: Gunakan garis bilangan. Pembuat nol adalah $x=3$ dan $x=-1$. Daerah yang memenuhi pertidaksamaan adalah di antara kedua titik tersebut.
Hasil: Fungsi turun pada interval $-1 < x < 3$.
2. (Kecukupan Data)
Berapakah nilai minimum dari fungsi $f(x)=ax^2+6x+c$?
(1) Titik stasioner fungsi terjadi di $x=-1$.
(2) Grafik fungsi melalui titik $(0, 10)$.
Analisis Soal: Untuk mencari nilai minimum, kita perlu tahu nilai fungsi di titik stasionernya. $f_{min}=f(x_{stasioner})$.
Analisis Pernyataan (1): $f'(x)=2ax+6$. Stasioner di $x=-1 \implies f'(-1)=2a(-1)+6=0 \implies -2a+6=0 \implies a=3$. Kita tahu bentuk fungsinya $f(x)=3x^2+6x+c$ dan titik minimumnya ada di $x=-1$. Tapi kita tidak tahu nilai minimumnya karena $c$ tidak diketahui. Jadi, (1) SAJA tidak cukup.
Analisis Pernyataan (2): Grafik melalui $(0,10) \implies f(0)=c=10$. Kita tahu $f(x)=ax^2+6x+10$. Kita tidak tahu di mana titik stasionernya karena $a$ tidak diketahui. Jadi, (2) SAJA tidak cukup.
Analisis Bersama-sama: Dari (1) kita dapat $a=3$. Dari (2) kita dapat $c=10$. Maka fungsi lengkapnya adalah $f(x)=3x^2+6x+10$. Nilai minimum terjadi di $x=-1$, yaitu $f(-1)=3(-1)^2+6(-1)+10 = 3-6+10=7$. Nilai minimum dapat ditentukan.
Hasil: Dua pernyataan BERSAMA-SAMA cukup. Jawaban C.

Bagian 4: Kunci Jawaban & Pembahasan

Pembahasan Lengkap (1-50)

Soal Mudah (1-10)

1. Jawaban: ATeori: Titik stasioner terjadi saat $f'(x)=0$.
Pengerjaan: $f'(x) = 2x-4$. $2x-4=0 \implies x=2$. Nilai stasionernya $f(2)=2^2-4(2)+7 = 4-8+7=3$. Jadi titiknya $(2,3)$.
2. Jawaban: BTeori: Fungsi naik saat $f'(x)>0$.
Pengerjaan: $f'(x) = -6-2x$. $-6-2x>0 \implies -2x>6 \implies x<-3$.
3. Jawaban: CTeori: Fungsi turun saat $g'(x)<0$.
Pengerjaan: $g'(x) = 3x^2-3$. $3x^2-3<0 \implies x^2-1<0 \implies (x-1)(x+1)<0$. Garis bilangan menunjukkan negatif antara -1 dan 1.
4. Jawaban: CTeori: Nilai minimum terjadi di titik stasioner.
Pengerjaan: $f'(x)=2x+8=0 \implies x=-4$. Nilai minimum $f(-4)=(-4)^2+8(-4)-1=16-32-1=-17$.
5. Jawaban: ATeori: Titik stasioner saat $f'(x)=0$.
Pengerjaan: $f'(x)=x^2-4x+3=0 \implies (x-1)(x-3)=0$. Jadi $x=1$ atau $x=3$.
6. Jawaban: CTeori: Fungsi turun saat $f'(x)<0$.
Pengerjaan: $f'(x)=-\sin x$. $-\sin x < 0 \implies \sin x > 0$. Ini terjadi di kuadran I dan II, yaitu $0 < x < \pi$.
7. Jawaban: ATeori: Titik stasioner saat $f'(x)=0$.
Pengerjaan: $f'(x)=-\cos x=0 \implies \cos x=0$. Untuk $0 \le x \le 2\pi$, $x=\pi/2$ dan $x=3\pi/2$. Salah satunya di $x=\pi/2$. Nilainya $f(\pi/2)=1-\sin(\pi/2)=1-1=0$. Titiknya $(\pi/2, 0)$.
8. Jawaban: BTeori: Gunakan uji turunan kedua.
Pengerjaan: $f'(x)=2(x-2)=0 \implies x=2$. $f''(x)=2$. Karena $f''(2)=2>0$, titik stasioner di $x=2$ adalah minimum lokal.
9. Jawaban: ATeori: Cek tanda turunan pertamanya.
Pengerjaan: $y' = \sec^2 x = \frac{1}{\cos^2 x}$. Karena $\cos^2 x$ selalu positif (dan tidak nol) pada interval $(0, \pi/2)$, maka $y'>0$. Jadi fungsi selalu naik.
10. Jawaban: CTeori: Nilai maksimum terjadi di titik stasioner.
Pengerjaan: $f'(x)=-2(x+1)=0 \implies x=-1$. Nilai maksimum $f(-1)=10-(-1+1)^2=10$.

17. Jawaban: BTeori: Nilai maks/min lokal terjadi di titik stasioner.
Pengerjaan: $y' = 1 - \frac{4}{x^2}=0 \implies x^2=4 \implies x=\pm 2$. Uji turunan kedua: $y''=\frac{8}{x^3}$. $y''(-2)=-1<0$ (maksimum). $y''(2)=1>0$ (minimum). Jadi maksimum di $x=-2$.

19. Jawaban: ATeori: Turun saat $f'(x)<0$.
Pengerjaan: $f'(x) = 2\cos x (-\sin x) = -\sin(2x)$. $-\sin(2x)<0 \implies \sin(2x)>0$. Untuk $0 \le x \le \pi$, maka $0 \le 2x \le 2\pi$. $\sin(2x)>0$ terjadi saat $0<2x<\pi$, yang berarti $0 < x < \pi/2$.

21. Jawaban: BTeori: Gunakan identitas sudut rangkap: $\sin(2A)=2\sin A \cos A$.

22. Jawaban: ETeori: Turun saat $f'(x)<0$. Domain fungsi adalah $x \ge 2$.
Pengerjaan: $f'(x) = 1\cdot\sqrt{x-2} + x \cdot \frac{1}{2\sqrt{x-2}} = \frac{2(x-2)+x}{2\sqrt{x-2}} = \frac{3x-4}{2\sqrt{x-2}}$. Pada domain $x \ge 2$, pembilang $3x-4$ selalu positif, dan penyebut juga positif. Maka $f'(x)>0$ selalu, jadi fungsi tidak pernah turun.
23. Jawaban: BTeori: Stasioner saat $f'(x)=0$.
Pengerjaan: $f'(x) = \frac{\cos x(2+\cos x)-\sin x(-\sin x)}{(2+\cos x)^2} = \frac{2\cos x + \cos^2 x + \sin^2 x}{(2+\cos x)^2} = \frac{2\cos x+1}{(2+\cos x)^2}$. $f'(x)=0 \implies 2\cos x+1=0 \implies \cos x = -1/2$. Solusinya $x=2\pi/3$ dan $x=4\pi/3$.
24. Jawaban: CTeori: Nilai ekstrim pada interval tertutup bisa terjadi di titik stasioner atau di ujung interval.
Pengerjaan: $f'(x)=3x^2-3=3(x-1)(x+1)=0$. Titik kritis di $x=1, x=-1$. Cek nilai di titik kritis dan ujung interval: $f(-2)=-1$, $f(-1)=3$, $f(1)=-1$, $f(3)=19$. Maka nilai maksimum adalah 19.
25. Jawaban: BTeori: Fungsi selalu naik jika $f'(x)\ge 0$ untuk semua $x$.
Pengerjaan: $f'(x) = k+\cos x$. Agar $f'(x) \ge 0$, kita perlu $k \ge -\cos x$. Karena nilai maksimum dari $-\cos x$ adalah 1 (ketika $\cos x = -1$), maka kita harus memilih $k$ yang lebih besar atau sama dengan nilai maksimum tersebut. Jadi, $k \ge 1$.

Soal Sulit (26-50)

26. Jawaban: CTeori: Optimisasi.
Pengerjaan: Misal bilangan $x$ dan $y$. $x+y=18 \implies y=18-x$. Fungsi yang dimaksimalkan $P = x^2y = x^2(18-x) = 18x^2-x^3$. $P' = 36x-3x^2 = 3x(12-x)=0$. $x=0$ (min) atau $x=12$. Jika $x=12$, maka $y=6$.
27. Jawaban: CTeori: Optimisasi.
Pengerjaan: Alas $s \times s$, tinggi $h$. Luas $L=s^2+4sh=27 \implies h=\frac{27-s^2}{4s}$. Volume $V=s^2h=\frac{s(27-s^2)}{4}$. $V'=\frac{27-3s^2}{4}=0 \implies s=3$. Maka $h=\frac{27-9}{12}=1.5$. $V=3^2(1.5)=13.5$.
28. Jawaban: Benar, Benar, Salah, BenarTeori: Kategori Benar/Salah.
Analisis: $f'(x)=2\cos(2x)$. Stasioner di $x=\pi/4, 3\pi/4$.
1. Pada $(0, \pi/4)$, $f'(x)>0$ (naik). Benar.
2. Pada $(\pi/4, \pi/2)$, $f'(x)<0$ (turun). Benar.
3. $f'(\pi/2)=2\cos(\pi)=-2 \ne 0$. Bukan stasioner. Salah.
4. Nilai stasioner $f(3\pi/4)=-1$ adalah minimum lokal. Benar.

32. Jawaban: CTeori: Cari titik stasioner.
Pengerjaan: $f'(x) = \cos x(1+\cos x) + \sin x(-\sin x) = 2\cos^2 x+\cos x-1=(2\cos x-1)(\cos x+1)=0$. Solusi stasioner $\cos x=1/2 \implies x=\pi/3$. Nilai maksimum $f(\pi/3) = \frac{\sqrt{3}}{2}(1+\frac{1}{2}) = \frac{3\sqrt{3}}{4}$.
33. Jawaban: ATeori: Syarat stasioner $f'(2)=0$.

34. Jawaban: BTeori: Analisis stasioner dan titik belok.
Pengerjaan: $y'=4x(x^2-a)$. Titik stasioner di $x=0, x=\pm\sqrt{a}$. Agar ada 3 titik stasioner, maka $a>0$. $y''=12x^2-4a$. Titik belok $y''=0 \implies x=\pm\sqrt{a/3}$. Agar ada 2 titik belok, maka $a>0$. Jika $a>0$, di $x=0$ adalah maks lokal, dan di $x=\pm\sqrt{a}$ adalah min lokal (ada 2 min). 36. Jawaban: CTeori: Stasioner di $x=1$ dan nilai $f(1)=-1$.

39. Jawaban: CTeori: Fungsi selalu turun jika turunan pertamanya, $f'(x)$, selalu lebih kecil atau sama dengan nol ($f'(x) \le 0$) untuk semua nilai $x$.
Pengerjaan: Turunan dari $f(x) = ax + \cos x$ adalah $f'(x) = a - \sin x$. Syarat selalu turun adalah $f'(x) \le 0$, maka $a - \sin x \le 0$, atau $a \le \sin x$. Agar pertidaksamaan ini berlaku untuk *semua* nilai $x$, nilai $a$ harus lebih kecil atau sama dengan nilai *minimum* yang mungkin dari $\sin x$. Nilai minimum dari fungsi $\sin x$ adalah -1. Oleh karena itu, haruslah $a \le -1$.

48. Jawaban: BTeori: Turunan logaritmik.
Pengerjaan: $\ln f = \frac{\ln x}{x}$. Turunkan: $\frac{f'}{f} = \frac{1-\ln x}{x^2}$. $f'=0 \implies 1-\ln x = 0 \implies x=e$.
49. Jawaban: ATeori: Anti-turunan (Integral).
Pengerjaan: $f'(x)=\int(6x-2)dx = 3x^2-2x+C_1$. $f'(1)=4 \implies 3(1)^2-2(1)+C_1=4 \implies C_1=3$. $f(x) = \int(3x^2-2x+3)dx = x^3-x^2+3x+C_2$. $f(1)=3 \implies 1-1+3+C_2=3 \implies C_2=0$. Jadi $f(x)=x^3-x^2+3x$. Maka $f(0)=0$.
50. Jawaban: CTeori: Definisi kecekungan dan nilai ekstrim.
Pengerjaan: $f'(c)=0$ dan $f''(x)>0$ untuk semua $x$. Karena $f''(c)>0$, titik tersebut adalah minimum lokal. Karena kurva selalu cekung ke atas ($f''(x)$ selalu positif), garis singgung di setiap titik selalu berada di bawah kurva. Garis singgung di $x=c$ adalah horizontal ($y=f(c)$), sehingga seluruh bagian kurva lainnya harus berada di atas garis ini. Ini berarti $f(c)$ adalah nilai terendah dari seluruh fungsi, yaitu minimum global.
