Bagian 4: Kunci Jawaban & Pembahasan

1. Jawaban: A Konsep: Barisan Geometri.
Diketahui $U_1=1$, $U_2=2$, $U_3=y$. Rasio $r = \frac{U_2}{U_1} = 2$.
Maka $y = U_2 \times r = 2 \times 2 = 4$.
Nilai $6-y = 6-4 = 2$.
2. Jawaban: D Konsep: Deret Geometri.
Barisan: $1, 2, 4, 8, 16, 32, 64, 128...$
Jumlah 5 suku pertama: $S_5 = \frac{1(2^5-1)}{2-1} = 31$.
Ditanya: $S_5 + 1 = 32$.
Diketahui $32 = \frac{1}{4} \times U_n \Rightarrow U_n = 128$.
Suku bernilai 128 adalah suku ke-8 ($1 \cdot 2^{8-1} = 128$).
3. Jawaban: C Konsep: Rata-rata Deret Geometri.
Barisan: $1, 2, 4...$
Rata-rata = $\frac{S_n}{n} = 3,75$.
$\frac{1(2^n-1)}{n} = \frac{15}{4}$. Uji nilai $n$: Jika $n=4$, $\frac{15}{4} = 3,75$. Benar.
4. Jawaban: E Konsep: Barisan Aritmetika.
$U_1=-2, U_4=-23$. Beda $b = \frac{-23 - (-2)}{4-1} = \frac{-21}{3} = -7$.
Barisan: $-2, -9, -16, -23$. Maka $p=-9, q=-16$.
$p-q = -9 - (-16) = 7$.
5. Jawaban: C Konsep: Sifat Suku Aritmetika.
$U_7 + U_8 = U_{10} + U_n$.
$(a+6b) + (a+7b) = (a+9b) + (a+(n-1)b)$.
$13b = 9b + (n-1)b \Rightarrow 4b = (n-1)b \Rightarrow n-1=4 \Rightarrow n=5$.
Jadi sama dengan suku ke-5.
6. Jawaban: B Konsep: Rata-rata Aritmetika.
Rata-rata $n$ suku pertama aritmetika = Suku Tengah ($U_{\text{tengah}}$) jika $n$ ganjil.
Rata-rata 9 suku = $U_5$.
$U_5 = a + 4b = -2 + 4(-7) = -2 - 28 = -30$.
7. Jawaban: E Konsep: Beda Aritmetika.
$U_3=6, U_4=2 \Rightarrow b = 2-6 = -4$.
$U_1=14, U_2=p-1$.
$U_2 = 14 + (-4) = 10 \Rightarrow p-1=10 \Rightarrow p=11$.
8. Jawaban: D Konsep: Persamaan Aritmetika.
$b=-4$. $U_7 = 14 + 6(-4) = -10$.
$U_{11} + U_n = 4 \cdot U_7 \Rightarrow (14+10(-4)) + U_n = 4(-10)$.
$-26 + U_n = -40 \Rightarrow U_n = -14$.
$U_n = a+(n-1)b \Rightarrow -14 = 14 + (n-1)(-4) \Rightarrow -28 = -4(n-1) \Rightarrow 7 = n-1 \Rightarrow n=8$.
9. Jawaban: A Konsep: Transformasi Data.
Barisan lama: $14, 10, 6...$ ($b=-4$). Rata-rata 10 suku = $\frac{U_1+U_{10}}{2}$.
$U_{10} = 14 + 9(-4) = -22$. Rata-rata lama = $\frac{14+(-22)}{2} = -4$.
Barisan baru dibagi 2, maka rata-rata baru = $\frac{-4}{2} = -2$.
10. Jawaban: D Konsep: Sudut Segiempat.
Total sudut segiempat $360^{\circ}$.
$\angle ADC = 360^{\circ} - (45^{\circ} + 120^{\circ} + 60^{\circ}) = 360^{\circ} - 225^{\circ} = 135^{\circ}$.
11. Jawaban: D Konsep: Trigonometri Dasar.
Tarik garis tinggi dari C ke F pada AB. Segitiga $CFB$ siku-siku di F. $\angle B = 120^{\circ}$ (tumpul), pelurusnya $60^{\circ}$ dalam segitiga imajiner, atau gunakan aturan sinus.
Dari soal: $\angle DCB=60^{\circ}$. Karena sejajar, tinggi $CF = BC \sin(180-120) = 4 \sin 60^{\circ} = 2\sqrt{3}$.
$DE = CF = 2\sqrt{3}$.
Segitiga $ADE$ siku-siku, $\angle A=45^{\circ}$. Maka $AE = DE / \tan 45^{\circ} = 2\sqrt{3}$.
12. Jawaban: E Konsep: Luas Bangun Datar.
$AFCD$ terdiri dari persegi panjang $EFCD$ dan segitiga $ADE$.
Luas $EFCD = DC \times DE = \sqrt{3} \times 2\sqrt{3} = 6$.
Luas $ADE = \frac{1}{2} \times AE \times DE = \frac{1}{2} \times 2\sqrt{3} \times 2\sqrt{3} = 6$.
Total = $6+6=12$.
13. Jawaban: D Konsep: Segitiga Sama Sisi & Jajar Genjang.
Segitiga ECF sama sisi, semua sudut $60^{\circ}$.
Di jajar genjang $ABCD$, $\angle C = 60^{\circ}$. Maka $\angle A = 60^{\circ}$.
$\angle B = 180^{\circ} - 60^{\circ} = 120^{\circ}$.
Dalam jajar genjang, $\angle EAD$ berseberangan atau sepihak? Perhatikan posisi titik.
Dari geometri, $\angle EAD = 120^{\circ}$.
14. Jawaban: C Konsep: Tinggi Segitiga Sama Sisi.
Sisi segitiga $EF = EA + AF = 5 + 4 = 9$.
Tinggi = $s \times \frac{1}{2}\sqrt{3} = \frac{9}{2}\sqrt{3}$.
15. Jawaban: B Konsep: Luas Segitiga Sama Sisi.
Luas = $\frac{1}{4} s^2 \sqrt{3}$.
$s=9 \Rightarrow \frac{81}{4}\sqrt{3}$.
16. Jawaban: E Konsep: Sudut Segitiga.
$\bigtriangleup ABC$ sama kaki. $\angle BED = 120^{\circ}$.
Perhatikan segiempat $BCDE$. Sudut pusat dan keliling atau jumlah sudut.
Hitungan detail: $\angle ABC = 75^{\circ}$.
17. Jawaban: B Konsep: Kesebangunan.
$\bigtriangleup ADE \sim \bigtriangleup ABF$.
$E$ tengah $AB$, maka $AB = 2AE$.
Perbandingan sisi: $\frac{BF}{DE} = \frac{AB}{AE} = 2$.
$BF = 2 \times 10 = 20$.
18. Jawaban: A Konsep: Luas Segitiga.
Luas $\bigtriangleup AEC = \frac{1}{2} \times AC \times DE$.
Kita perlu cari $AC$. Dari soal 17, $BF=20$. Sudut A bisa dicari.
Atau gunakan $AB=AC$. Dari $\triangle ABF$, $AF = BF/\tan A$.
Dari sudut $120^{\circ}$ di $E$, pelurusnya $60^{\circ}$. $\triangle ADE$ siku-siku, $\angle A = 30^{\circ}$.
$AC = AB = BF / \sin 30^{\circ} = 20 / 0.5 = 40$.
Luas = $\frac{1}{2} \times 40 \times 10 = 200$.
19. Jawaban: B Konsep: Titik Potong Fungsi.
$g(1)=0 \Rightarrow 2-b(1)=0 \Rightarrow b=2$.
$g(x) = 2-2x$, $h(x) = 1-2x+x^2$.
Potong: $2-2x = 1-2x+x^2 \Rightarrow x^2-1=0 \Rightarrow x=\pm 1$.
Jika $x=-1$, $y = 2-2(-1) = 4$. Titik $(-1, 4)$.
20. Jawaban: C Konsep: Integral Tentu.
$\int ( (2-2x) - (1-2x+x^2) ) dx = \int (1 - x^2) dx$.
$= x - \frac{1}{3}x^3 + C$.
21. Jawaban: C Konsep: Luas Daerah Integral.
Batas integrasi dari titik potong $x=-1$ sampai $x=1$.
$\int_{-1}^{1} (1-x^2) dx = [x - \frac{1}{3}x^3]_{-1}^{1}$.
$= (1-1/3) - (-1+1/3) = 2/3 - (-2/3) = 4/3$.
22. Jawaban: B Konsep: Persamaan Kuadrat.
$x^2+x-2 = x+2 \Rightarrow x^2-4=0 \Rightarrow x=\pm 2$.
Absis salah satunya -2.
23. Jawaban: C Konsep: Integral Aljabar.
$\int ((x+2) - (x^2+x-2)) dx = \int (4-x^2) dx$.
$= 4x - \frac{1}{3}x^3 + C$.
24. Jawaban: C Konsep: Luas Daerah.
Batas -2 sampai 2.
$\int_{-2}^{2} (4-x^2) dx = [4x - \frac{1}{3}x^3]_{-2}^{2}$.
$= (8 - 8/3) - (-8 + 8/3) = 16 - 16/3 = 32/3$.
25. Jawaban: D Konsep: Titik Potong.
$-2x+10 = (x^2-4x+4)-2 \Rightarrow -2x+10 = x^2-4x+2$.
$x^2-2x-8=0 \Rightarrow (x-4)(x+2)=0$. $x=4, -2$.
$y$ saat $x=4 \to g(4) = 2$. $y$ saat $x=-2 \to g(-2)=14$.
Terkecil adalah 2.
26. Jawaban: B Konsep: Integral.
$g(x)-f(x) = (x^2-4x+2) - (-2x+10) = x^2-2x-8$.
$\int (x^2-2x-8) dx = \frac{1}{3}x^3 - x^2 - 8x + C$.
27. Jawaban: D Konsep: Luas Daerah (Sebelah Kanan Sumbu Y).
Batas dari $x=0$ sampai $x=4$ (titik potong positif).
$\int_{0}^{4} (f(x)-g(x)) dx = \int_{0}^{4} (-x^2+2x+8) dx$.
$= [-\frac{1}{3}x^3 + x^2 + 8x]_0^4 = -64/3 + 16 + 32 = 48 - 21.33 = 80/3$.
28. Jawaban: B Konsep: Algoritma/Flowchart.
$x=7 \to x^2=49 > 40$ (Tidak).
$y = x+4 = 11$.
$z = 2y = 22$.
$z$ habis dibagi 4? (Tidak).
$w = (5z-6)/4 = (110-6)/4 = 104/4 = 26$.
(1) $y-2x = 11-14 = -3$ (Ganjil/Benar).
(2) $w < 20$ (26 < 20 Salah).
(3) $x \cdot y = 77$ (Ganjil/Benar).
(4) $w$ kelipatan 3 (26 bukan kelipatan 3/Salah).
Benar: (1) dan (3).
29. Jawaban: E Konsep: Algoritma.
$x=7$. $y=x+3=10$.
$y$ genap? Ya. $z=2x-y = 14-10=4$.
$z < y$? (4 < 10 Ya).
$w = z/2 = 2$.
Semua pernyataan (prima, genap, <7, faktor 10) Benar.
30. Jawaban: E Konsep: Algoritma.
$x=7$. $x$ kelipatan 2? Tidak.
$y = 2x-1 = 13$.
$y$ genap? Tidak.
$z = 5y-10 = 65-10 = 55$.
(1) $z>50$ (Benar).
(2) $z$ habis dibagi 5 (Benar).
(3) $x+y+z = 7+13+55 = 75$ (Ganjil/Benar).
(4) $xy = 91$ (Ganjil/Benar).
