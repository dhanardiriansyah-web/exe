Pendahuluan: Pertidaksamaan

Pertidaksamaan (pertaksamaan) merupakan kalimat terbuka (kalimat yang belum tentu nilai kebenarannya), sedangkan ketidaksamaan merupakan kalimat tertutup. Soal-soal pertidaksamaan tujuannya secara umum adalah mencari batas nilai variabel agar kalimat bernilai benar.

Teori Dasar Pertidaksamaan

  1. Penjumlahan/Pengurangan: Jika $a \leq b$ maka $a+c \leq b+c$ (Tanda Tetap).
  2. Perkalian/Pembagian Positif: Jika $a \leq b$ dan $c > 0$, maka $a \times c \leq b \times c$ (Tanda Tetap).
  3. Perkalian/Pembagian Negatif: Jika $a \leq b$ dan $c < 0$, maka $a \times c \geq b \times c$ (Tanda Berubah).

Jenis Pertidaksamaan Umum

  • Pertidaksamaan Linear: $ax+b \leq 0$
  • Pertidaksamaan Kuadrat: $ax^{2}+bx+c \leq 0$
  • Pertidaksamaan Pecahan: $\dfrac{f(x)}{g(x)} \leq 0$ dimana $g(x) \neq 0$
  • Pertidaksamaan Bentuk Akar: $\sqrt{f(x)} \leq 0$ dimana $f(x) \geq 0$
  • Pertidaksamaan Harga Mutlak: $|f(x)| \leq 0$ dimana $|f(x)|=\sqrt{f^{2}(x)}$

Pertidaksamaan Eksponen & Logaritma

Eksponen:

  • Untuk $a > 1$: Jika $a^{f(x)} \leq a^{g(x)}$ maka $f(x) \leq g(x)$ (Tanda Tetap)
  • Untuk $0 < a < 1$: Jika $a^{f(x)} \leq a^{g(x)}$ maka $f(x) \geq g(x)$ (Tanda Berubah)

Logaritma:

  • Untuk $a > 1$: Jika ${}^a\!\log f(x) \leq {}^a\!\log g(x)$ maka $f(x) \leq g(x)$
  • Untuk $0 < a < 1$: Jika ${}^a\!\log f(x) \leq {}^a\!\log g(x)$ maka $f(x) \geq g(x)$
