Bagian 5: Kunci Jawaban & Pembahasan

1. Jawaban: B Konsep: Akar Persamaan Kuadrat
Jika akar-akar $x^{2}- \left( 3a-5 \right)x+3=0$ dimisalkan $m$ dan $n$ dimana $m=3n$:
Hasil kali akar: $m \cdot n = \frac{c}{a} \Rightarrow m \cdot 3m = 3 \Rightarrow 3m^2=3 \Rightarrow m=\pm 1$.
Hasil jumlah: $m+n = -\frac{b}{a} \Rightarrow 4m = 3a-5$.
Jika $m=-1 \Rightarrow -4=3a-5 \Rightarrow a=1/3$.
Jika $m=1 \Rightarrow 4=3a-5 \Rightarrow a=3$.
Perkalian nilai $a$: $1/3 \times 3 = 1$.
2. Jawaban: D Konsep: Sifat Akar-Akar
$x^{2}+px+q=0$ akarnya $p$ dan $q$.
$p \cdot q = q \Rightarrow p=1$.
$p+q = -p \Rightarrow 1+q=-1 \Rightarrow q=-2$.
$p^2-2q = 1^2 - 2(-2) = 1+4 = 5$.
3. Jawaban: C Konsep: Selisih Akar
Rumus selisih akar: $|x_1-x_2| = \frac{\sqrt{D}}{a}$.
$1 = \sqrt{(2a)^2 - 4(1)(4/3a)}$.
$1 = 4a^2 - \frac{16}{3}a \Rightarrow 12a^2 - 16a - 3 = 0$.
Faktor: $(6a+1)(2a-3)=0 \Rightarrow a=-1/6$ atau $a=3/2$.
Soal meminta selisih $a$ dan $4/6$. Untuk $a=3/2$: $3/2 - 4/6 = 9/6 - 4/6 = 5/6$.
4. Jawaban: C Konsep: Nilai Maksimum Fungsi
Misal $M = x_1^2 + x_2^2 = (x_1+x_2)^2 - 2x_1x_2$.
$M = (\frac{2c-1}{2})^2 - 2(\frac{-c^3+4}{2}) = c^3+c^2-c - 15/4$.
Turunan $M' = 3c^2+2c-1 = 0 \Rightarrow (3c-1)(c+1)=0$. $c=1/3, -1$.
Uji turunan kedua, $M'' = 6c+2$. Saat $c=-1$, $M'' < 0$ (Maksimum).
Substitusi $c=-1$ ke $M$: $-1 + 1 - (-1) - 15/4 = 1 - 3.75 = -2.75 = -2\frac{3}{4}$.
5. Jawaban: B Konsep: Aplikasi Turunan pada PK
$N = x_1+x_2+x_1x_2 = -5a + (a^3-4a+1) = a^3-9a+1$.
Turunan $N' = 3a^2-9=0 \Rightarrow a^2=3 \Rightarrow a=\pm \sqrt{3}$.
Cek interval $[-3,3]$. Nilai maksimum tercapai saat $a=-\sqrt{3}$.
6. Jawaban: E Konsep: Bilangan Bulat & PK
$x_1+x_2 = 6$ dan $x_1x_2 = q$.
Pasangan bulat positif $(x_1, x_2)$ yang jumlahnya 6: $(1,5), (2,4), (3,3)$.
$q$ yang mungkin: $1\cdot5=5, 2\cdot4=8, 3\cdot3=9$.
Jumlah semua $q$: $5+8+9=22$.
7. Jawaban: B Konsep: Diskriminan & Selisih Akar
Selisih akar $= \frac{\sqrt{D}}{a} = 3$.
$\frac{\sqrt{(-10a)^2 - 4(5)(8a)}}{5} = 3 \Rightarrow \sqrt{100a^2-160a} = 15$.
$100a^2-160a = 225 \Rightarrow 20a^2-32a-45=0$.
Faktor: $(2a-5)(10a+9)=0$. Karena $a>0$, maka $a=5/2$.
$a^2-a = (5/2)^2 - 5/2 = 25/4 - 10/4 = 15/4 = 3\frac{3}{4}$.
8. Jawaban: A Konsep: Substitusi Variabel
Misal $y = x^2-10x$. Persamaan: $\frac{1}{y-29} + \frac{1}{y-45} - \frac{2}{y-69} = 0$.
Setelah disederhanakan, ditemukan $y=39$.
Kembalikan ke $x$: $x^2-10x = 39 \Rightarrow x^2-10x-39=0$.
Perkalian akar $x_1x_2 = c/a = -39$.
9. Jawaban: B Konsep: Operasi Akar
Jumlah akar: $-(b+1) = -a^2 \Rightarrow b+1=a^2$.
Kali akar: $a^2+5 = b^2$. Substitusi $a^2$: $(b+1)+5=b^2 \Rightarrow b^2-b-6=0$.
$(b-3)(b+2)=0$. Karena $b>0 \Rightarrow b=3$.
$a^2 = 3+1=4 \Rightarrow a=2$ (karena $a>0$).
$a+b-ab = 2+3-(2)(3) = 5-6 = -1$.
10. Jawaban: A Konsep: Identitas Aljabar
$\frac{1}{r^2} + \frac{1}{s^2} = \frac{s^2+r^2}{(rs)^2} = \frac{(r+s)^2-2rs}{(rs)^2}$.
$= \frac{(-b/a)^2 - 2(c/a)}{(c/a)^2} = \frac{b^2/a^2 - 2ac/a^2}{c^2/a^2}$.
$= \frac{b^2-2ac}{c^2} = \frac{(b^2-4ac)+2ac}{c^2} = \frac{D+2ac}{c^2} = \frac{D}{c^2} + \frac{2a}{c}$.
11. Jawaban: D Konsep: Identitas Aljabar
$x_1^2 + x_2^2 = (x_1+x_2)^2 - 2x_1x_2$.
$10 = (-2a)^2 - 2(b^2) \Rightarrow 10 = 4a^2 - 2b^2$.
$2b^2 = 4a^2 - 10 \Rightarrow b^2 = 2a^2 - 5$.
12. Jawaban: A Konsep: Substitusi Akar
Jumlah akar: $p + (p^2-2) = -(-10) = 10 \Rightarrow p^2+p-12=0 \Rightarrow (p+4)(p-3)=0$.
$p>0 \Rightarrow p=3$. Akar-akar: $3$ dan $7$. $c = 3 \cdot 7 = 21$.
$c=21$ akar dari $x^2+ax+42=0 \Rightarrow 21^2 + 21a + 42 = 0$.
$21a = -483 \Rightarrow a = -23$.
13. Jawaban: A Konsep: Substitusi Variabel
Misal $y=\sqrt[3]{x}$. $y(1+y)=2 \Rightarrow y^2+y-2=0 \Rightarrow (y+2)(y-1)=0$.
$y=-2 \Rightarrow x = (-2)^3 = -8$.
$y=1 \Rightarrow x = (1)^3 = 1$.
Hasil kali: $(-8) \cdot 1 = -8$.
14. Jawaban: E Konsep: Sifat Akar Berkebalikan
Akar $m$ dan $1/m$. Kali akar: $m \cdot 1/m = q \Rightarrow q=1$.
Jumlah: $m + 1/m = p$. Karena akar negatif, misal $m=-k (k>0)$.
$p = -(k+1/k)$. Nilai min $k+1/k$ adalah 2 (saat $k=1$). Maka max $p = -2$.
Nilai max $p-q = -2 - 1 = -3$.
15. Jawaban: B Konsep: Operasi Akar
$\frac{1}{x_1}+\frac{1}{x_2} = \frac{x_1+x_2}{x_1x_2} = \frac{-q/p}{-1/p} = q = -1$.
$x_1x_2 = -1/p \Rightarrow (-1.5x_2)x_2 = -1/p \Rightarrow x_2^2 = \frac{2}{3p}$.
$x_1+x_2 = -q/p = 1/p$. $-0.5x_2 = 1/p \Rightarrow x_2 = -2/p$.
$(-2/p)^2 = 2/3p \Rightarrow 4/p^2 = 2/3p \Rightarrow p=6$.
$p+q = 6 + (-1) = 5$.
16. Jawaban: E Konsep: Persamaan Kuadrat
$\frac{q+1}{p+1} - \frac{p-1}{q-1} = -\frac{3}{2}$.
Setelah disederhanakan (lihat pembahasan detail di soal asli), persamaan dalam $k$: $k^2+2k-8=0$.
Jumlah nilai $k = -2$.
17. Jawaban: D Konsep: Persamaan Kuadrat Baru
$\frac{1}{\alpha^2} + \frac{1}{\beta^2} = 5 \Rightarrow p^2-4=5 \Rightarrow p=3$ (karena $p>0$).
$\frac{1}{\alpha^2\beta^2} = q \Rightarrow q = 1/(1/2)^2 = 4$.
Nilai $q-p = 4-3 = 1$.
18. Jawaban: A Konsep: Substitusi Akar
$a^2 = 3-a$ dan $b^2 = 3-b$.
$2a^2+b^2+a = 2(3-a) + (3-b) + a = 6-2a+3-b+a = 9-(a+b)$.
$a+b=-1$, jadi $9-(-1) = 10$.
19. Jawaban: C Konsep: Transformasi Akar
Jumlah akar baru: $(m+2)+(n+2) = (m+n)+4 = -b/a+4$.
$-q/a = -b/a+4 \Rightarrow q = b-4a$.
Kali akar baru: $(m+2)(n+2) = mn+2(m+n)+4 = c/a - 2b/a + 4$.
$r/a = c/a - 2b/a + 4 \Rightarrow r = c-2b+4a$.
$q+r = b-4a + c-2b+4a = c-b$.
20. Jawaban: A Konsep: PK Baru
$x_1+x_2=-3, x_1x_2=1$.
Akar baru $y_1, y_2$. $y_1+y_2 = 4 + \frac{x_1^2+x_2^2}{x_1x_2} = 4 + \frac{(-3)^2-2}{1} = 11$.
$y_1y_2 = (2+\frac{x_2}{x_1})(2+\frac{x_1}{x_2}) = 4 + 2(\frac{x_1^2+x_2^2}{x_1x_2}) + 1 = 5 + 2(7) = 19$.
$x^2 - 11x + 19 = 0$.
